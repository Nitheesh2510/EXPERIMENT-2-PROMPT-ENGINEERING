# EXP-2-PROMPT-ENGINEERING-

## Aim: 
Comparative Analysis of different types of Prompting patterns and explain with Various Test Scenarios

Experiment:
Test and compare how different pattern models respond to various prompts (broad or unstructured) versus basic prompts (clearer and more refined) across multiple scenarios. 
Analyze the quality, accuracy, and depth of the generated responses.


## Algorithm:
### Step 1: Start the Experiment
Begin the comparative analysis of prompting patterns.

### Step 2: Define the Objective
- Compare different prompting techniques.
- Analyze effectiveness in generating AI responses.
- Measure quality, accuracy, and response depth.

### Step 3: Select Prompting Patterns
Choose different prompt types for analysis:
- Broad/Unstructured Prompting
- Basic/Refined Prompting
- Zero-Shot Prompting
- Few-Shot Prompting
- Chain-of-Thought Prompting

### Step 4: Choose Test Scenarios
Prepare multiple test cases such as:
- Mathematical reasoning
- Structured data extraction
- Ethical reasoning
- General knowledge question answering

### Step 5: Design Prompts
- Create prompts for each scenario.
- Ensure the same task is tested using all prompting patterns.

### Step 6: Input Prompts into AI Model
- Enter each prompt into the AI system.
- Generate responses for all prompt types.

### Step 7: Collect Generated Responses
- Save all outputs.
- Organize responses based on:
  - Prompt type
  - Test scenario

### Step 8: Evaluate Responses
Analyze outputs using:
- Accuracy
- Relevance
- Clarity
- Depth of explanation
- Consistency
- Token usage/cost

### Step 9: Compare Results
- Compare broad prompts with refined prompts.
- Compare Zero-Shot, Few-Shot, and Chain-of-Thought methods.

### Step 10: Analyze Findings
Observe:
- Strengths of each prompting pattern
- Weaknesses of each prompting pattern
- Impact of prompt structure on AI output

### Step 11: Draw Conclusion
Conclude which prompting pattern performs best based on:
- Response quality
- Accuracy
- Efficiency

### Step 12: Stop
End the experiment.
